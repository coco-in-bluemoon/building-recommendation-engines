# 1. 추천 엔진 설계 과정 (사용자 기반 Collaborative Filtering
- 데이터를 불러온다.
- 사용자 간의 유사도를 계산한다. 2장에서는 상관계수를 사용한다.
- 사용자 아이디가 주어졌을떄, 해당 사용자가 시청하지 않은 영화를 대상으로 Rating을 계산한다.
  - [다른 사용자와의 유사도(상관계수) * 다른 사용자의 Rating]의 평균
- 추론한 Rating과 결과를 반환한다.
